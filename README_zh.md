# Typer
Typer 是一款打字机输出模拟器，让你的 log 输出更加生动！

![Shown](doc/show.gif)


# 立即体验！
## 先决条件
在您的个人电脑上已经安装 [Go](https://go.dev/doc/install) 和 [Git](https://git-scm.com/downloads)。

## 将 Typer 仓库克隆到您本机
```shell
git clone https://github.com/ZenkieBear/typer
```

## 运行 `main.go`
```shell
go run main.go
```
